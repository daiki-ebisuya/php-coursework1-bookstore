# PHP Coursework 1 – Online Bookstore

## Project Name
Online Bookstore (PHP Coursework 1)

## Short Description
This project is a simple online bookstore built in PHP as the final assignment for Week 1.  
It demonstrates the use of associative arrays, functions, conditionals, loops, POST data handling,  
server metadata, date/time functions, file logging, and basic HTML output formatting.

This application allows users to:
- View a list of books
- Apply genre-based discounts
- Add new books through a POST form
- Automatically log each new addition to a log file
- Display server information (timestamp, IP, user agent)

  Bonus feature shows:
- Additional genre-based discounts
- A preview of the log file using `file_get_contents()`.
